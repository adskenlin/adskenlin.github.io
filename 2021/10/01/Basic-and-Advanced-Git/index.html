Basic and Advanced Git
Post：2021-10-01 03:01:30 Tags：/ Git / Visit：

Git is version-control tool for tracking changes in files. It is widely used in software development project and It significantly increases the working efficient among teams and supports for distributed, non-linear workflows (thousands of parallel branches running on different systems). One thing more to mention, it was created by Linus Torvals (Genius!!!).

This Blog is written for marking down the key points and tricks of git. It will keep updating!

Installation

Follow the official websites to install git: https://git-scm.com/book/en/v2/Getting-Started-Installing-Git

To verify the installation, use git --version in terminal, you should see the output git version x.xx.x.
